# Training on Google Colab

To train this model on Google Colab:

1. **Prepare your Data**: 
   - Create a folder called `data` inside `ASR_project`.
   - Put your `wav` folder and `lab` folder directly inside that `data` folder.
   - It should look like this: `ASR_project/data/wav` and `ASR_project/data/lab`.

2. **Upload** the entire `ASR_project` folder to your **Google Drive**.

3. **Open** [asr_training.ipynb](asr_training.ipynb) in Google Colab and follow the steps (Mount Drive -> Install -> Train).
